服务器规则

为了维护良好的游戏环境，请遵守以下规则

基本规则

  • 尊重其他玩家，禁止使用侮辱性语言
  • 禁止作弊、使用外挂或任何不公平的游戏方式
  • 请勿在游戏中进行任何形式的广告宣传
  • 禁止破坏其他玩家的建筑，除非得到允许
  • 请勿滥用游戏机制或寻找漏洞

行为准则

  • 保持聊天内容适合所有年龄段
  • 请勿过度刷屏或使用全大写字母
  • 尊重管理员的决定，如有异议请礼貌沟通
  • 禁止建造不当或冒犯性的建筑
  • 不要在游戏中索要物品或OP权限

违规处罚

违反上述规则将根据情节轻重给予相应处罚，包括警告、禁言、临时封禁和永久封禁。

管理员保留对未列出的不当行为进行处罚的权利。如果你对处罚有异议，可以通过论坛或 Discord 联系我们。

加入我们

准备好开始你的冒险了吗？按照以下步骤加入我们的服务器

1. 下载游戏

首先需要下载并安装 Minecraft Java Edition。你可以从 Mojang 官方网站购买并下载游戏。

访问官网

2. 添加服务器

打开游戏，点击"多人游戏"，然后点击"添加服务器"。在服务器地址栏中输入：

mc.wzyserver.com

3. 开始游戏

添加成功后，选择我们的服务器并点击"加入服务器"。等待加载完成，开始你的冒险之旅！

系统要求

最低配置

  • 操作系统: Windows 7/8/10, macOS 10.12+, Linux
  • 处理器: Intel Core i3/AMD equivalent
  • 内存: 4 GB RAM
  • 显卡: Intel HD Graphics 4000
  • 存储空间: 1 GB 可用空间

推荐配置

  • 操作系统: Windows 10, macOS 10.13+, Linux
  • 处理器: Intel Core i5/AMD equivalent
  • 内存: 8 GB RAM 或更高
  • 显卡: NVIDIA GeForce GTX 960 或更高
  • 存储空间: 2 GB 可用空间
